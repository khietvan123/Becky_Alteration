# Becky Alteration Website

A modern, high-converting website built for **Becky Alteration**, a local clothing alteration service in Mississauga, Ontario.

The website is designed to showcase services, build trust through reviews, and convert visitors into leads using a clean UI and a functional contact form.

---

## 🚀 Live Features

- ✨ Elegant, responsive UI (mobile + desktop)
- 📩 Contact form with email integration (EmailJS)
- ⭐ Social proof section (Google reviews style)
- 📍 Google Maps integration
- 🎯 Conversion-focused design (clear CTA: *Get a Free Quote*)
- ⚡ Fast performance with Vite + React

---

## 🧰 Tech Stack

- **Frontend:** React + TypeScript
- **Build Tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** Lucide React
- **Email Service:** EmailJS
